# Sistema-Web-Acos-Maq
Sistema web interno para a Aços Maq, desenvolvido para apoiar o setor de vendas no cadastro de clientes, criação de orçamentos e gestão de pedidos no ramo de ferros e aços. O projeto utiliza HTML, CSS, JavaScript e PHP, com foco em organização comercial, agilidade nos processos e armazenamento centralizado das informações.
